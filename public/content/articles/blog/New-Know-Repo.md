# 关于新的知识库

我终于把知识库变成线上的了(当然仰赖了一些数字生命的帮助，毕竟老风说我的代码技术并不行).

不过当我可以让我的两台电脑以1分钟不到的延时同步的时候,我是不是真正地具有代码能力并不重要,我实现了我的目的.总而言之,也算部分实现了 Linux Torvalds 说的:**我没存储焦虑,我把所有的内容放在 Internet 上,如果它有用会有人替我存储的.**

知识库的目的并不是抄书,而是把自己的认识记录下来.从这个角度来说,博客和知识库的内核是相似的.区别~是博客有背景音乐~是博客更多是记录一些我的大事记,或者是我那100字无法记录的吐槽.

正如老风说的,他很难理解我居然一口气能写那么多字,从这个角度来说我的更新其实也并不少.或许是因为我周围的大神比较多,或者是周围给我添堵的人比较多,或者是我这个人记录的意愿比较高涨,总而言之我是有很多话想说啰里啰唆的但是你都开始看了就这样吧我也并不是很想改.毕竟这是我的部落格.

不管怎么说,知识库的上线总是一个里程碑.我总是跨越了知识私人化的鸿沟,真正地让你们看到我的思维.理不理解我都无所谓.这个世界上有不少的天才不能理解我这个凡人的思维.这个世界上也有不少的蠢猪不能看透的我的想法.总之我对所有人都是平等地不在乎,当然除了我的家人.

或许我们都应该把自己的认识写下来,至少从数学的角度我们应该这么做.你应该会使用你的知识,而不是仅仅把它记下来.有些人学完莫迪图还不会设计水泵,那么学习的意义是什么?考试100分如何呢.

注意到我莫名其妙加入了一个科研小组.这个小组里的绝大部分人我都认识,这很奇特.但是认识的人都值得吐槽两句,不过不应该放在博客的环节吐槽.

或许我写的文字前后逻辑很跳跃,比如这篇文章应该是说我的知识库的,然后我就锐评了一下知识的获取.不管怎么样,你可以访问https://github.com/rougamorika/MyThoughtsVault/
当然我不喜欢你修改我的绝大部分东西,也不奢求你理解绝大部分的东西,毕竟绝大部分的时候我都不知道我的那些个想法在做什么.但是不论怎么样我还是把我的大脑的一小部分给你看了一眼.

当然我不是专业的,这也不是百科全书.我是很反对把知识库变成百科全书的,这世界上已经有一个很重要的东西叫wikipedia了.总之,想法比知识本身要重要的多.但是很显然科研小组里大家只在乎名字有没有被改成合适的格式.

Fuck it.
