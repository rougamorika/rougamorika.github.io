计数基本法则

我们现在假设有[汉堡,饼干,水果]三种食物和[可乐,雪碧]两种饮料,组成套餐有几种情况呢??这样的计数问题在数学中叫做组合分析. 对于上面的问题,我们一共能组成种方案.我们发现.如果多加一种饮料,那么我们的配对方法就变成了.假如我们有种主食和种饮料,那么我们能配出种套餐.这是因为,我们以主食为参考对象(以饮料也是一样的),每一种主食都能和剩下的所有饮料组成一种组合,所以一种主食就有种组合方式,那么种主食就会有种组合方式.我们可以看图来认识,下面是一个配对图

flowchart TD
A(汉堡) -->|套餐1| D(可乐)
A -->|套餐2| E(雪碧)
B(饼干) -->|套餐3| D
B -->|套餐4| E
C(水果) -->|套餐5| D
C -->|套餐6| E

我们现在假设我们买到的套餐还能附赠一种自选的玩具,那么我们又会有多少中可能呢?我们可以把先前的6个套餐看成一个整体,然后就将三种物品的计数化归成了两种物品的计数,那么我们我就可以重新用已有的方法来处理课,所以我们发现总的可能数字就是每一步选项数目的乘积.也就是说,假设我们有个步骤,每个步骤有种类方案,那么我们就应该用 每一步内的方案数应该用加法,因为我们一步之内的行为是在分类,我们数种类用的是加法,所以每一个的确定要通过加法,这叫做加法原理.步与步之间的处理通过我们的分析应当用乘法,称为乘法原理.

方法论(1)-5岁理论与自我教学法

发表于 2024-10-23 分类于 其他-方法论
本文字数： 1.8k 阅读时长 ≈ 2 分钟

什么是名词党

我就是.所以我有足够的理由和大家分享名词党的经验,想要成为初阶的名词党,只要背熟数学辞海中的词条就可以了.这时候的重点是控制住自己的分享欲,以及拥有及时转移话题的能力,这样不至于让懂行的人过多追问. 高阶的名词党知道要建立一些背景,比如买下一些某些领域的名著,比如买下一本卓里奇.然后不时地拿出来看看,一定要找有人的时候拿出来,不能没人的时候拿出来,要积极地去抓那些初阶的名词党,甚至以抓初阶的名词党为乐 活用”我好菜”,“你是大佬”,“我略懂数理”这样的卖菜语句即可.

我不想当名词党了

名词党一定程度上是利用了数学的高门槛,建立起虚荣心的一种行为,简单地说就是爱装逼.正如算法大佬Empty_Dust所说,有的时候我(指笔者本人)什么都不懂,还要顾左右而言它.过去我并不觉得这是什么很大的问题,甚至觉得这是串联知识的”象征”. 实际上,知识深度的广泛连接要远远地重要于表面上知识之间的连接,所谓的名词和名词的链接.我的评价是这种链接就是他妈的傻逼.
于是,我写下这篇文章,批评自己,也是劝谏大家不要当名词党.

如何摆脱名词的追击??:5岁理论和遗忘曲线

这句话就是变相地在问:怎么深入理解一个/多个知识点.答案是把它教给一个5岁的孩子.你如果能把你现在所学的内容教会一个完全不懂这个理论的,5~8岁的刚刚建成逻辑能力的孩子,那么你可以称得上理解通透了这个知识点.实际上,这个五岁的孩子并不需要是一个实际的人,它可以是你的另一个学习”人格”.这个”人格”不接受一切空泛的论述,一切都是例子,本质,便于理解嚼碎了的知识.这种咀嚼的过程能够让你深入理解知识.
为了让一个5岁的懂得很多抽象的东西,有的时候仅仅通过文字是不够的,而且当我们接受了很多文本之后,有时候还不如一个五岁孩子,这时候用图.孩子们看图都很方便,爱看插图.为什么我们不能这么做呢.

flowchart TD
    A[我不想当名词党] -->|问题| B(我把握不住知识重点)
    B --> C{解决方法}
    C -->|我的认识空虚| D[简化知识,活用类比/图像]
    C -->|我虚荣| E[转换心态,主动深入]
    C -->|我只能记住名词| F[认识遗忘曲线,合理计划]

当我们这样写内容时,我们就更能抓住这段文本的主线,便于理解.而且我们通过相邻的部分还可以帮助我们梳理/推测遗忘的部分.比如我们根据这个流程图,我们的第三个部分就是讲遗忘.现在我们来看一张图,它一定程度反映了大部分人的记忆情况: 我们活用图像,不止是学会作图,更重要是图上捕捉内容,比如我们根据Ebbinghaus遗忘曲线:

  • 我们的短期记忆的消失速度很快,差不多20分钟
  • 1天我们就已经只剩约3成的记忆了
  • 5天之后我们就已经忘记的差不多了

我们怎么样吗面对这样的”侵蚀”?我们有第二张图像,同样也是这个研究的一个部分: 这个研究的结果表明:

  • 随着复习次数的增加,遗忘的速度会减慢
  • 1,3,7的间隔是一个很好的复习模式
  • 我们能记住9成以上就很不错!!

实际上,由于我们学习的是理科,我个人的经验是85%的记忆+15%的泛化,所谓的泛化就是指创新地思考,建立许多有趣/看起来没意义的联系,而不是对我们的知识点完全忠实,就事论事.也就是说,锤子不一定只能是锤钉子的,还可以是武器,改装一下还可以用来起钉子.我们要保留这种泛化,“空想”的能力.
所以我们现在合在一起想,那么我们就得出了我们的结论:那就是每隔一段时间自己教自己.我称之为自我教学.因为我们不可能把每一个知识点都立刻变成5岁孩子能理解的状态,而且我们无法保证自己的理解一定是正确的/完善的.所以我们要不断地复习,纠错,修改我们的知识体系,就像建造一幢大楼,我们可以画一张图

flowchart LR
A[新的知识点] -->|简化/类比| B(第一阶段知识)
B -->|慢慢遗忘/出错|A
B -->|修正/理解/多次复习|C{不容易被遗忘的知识}
C -->|动脑子/结合其他知识|A

其中第一阶段知识就是指经过一天忘超过25%的知识,不容易忘的知识就经过3天忘记不超过10%的知识.

总结:我只是想学更多的知识

知识是学不完的.庄子有云:“吾生也有涯，而知也无涯。以有涯随无涯，殆已！已而为知者，殆而已矣！为善无近名，为恶无近刑，缘督以为经，可以保身，可以全生，可以养亲，可以尽年”(《庄子·内篇·养生主第三》)
许多人就截取第一句,然后高呼没必要学知识,躺平即可.对此我的评价是,你慢慢睡,我慢慢学.
